<h1>Maintain the CheckBox state of GridView when pagingin in ASP.NET (CSASPNETPagingGridViewMaintainCheckBox)</h1>
<h2>Introduction</h2>
<p class="MsoNormal">This demo is mainly showing you how to make the CheckBox in the each row of GridView &quot;keep its state&quot; when paging.<span style="">
</span></p>
<h2>Running the Sample</h2>
<p class="MsoNormal">Step 1: Open the CSASPNETPagingGridViewMaintainCheckBox.sln.<br>
Step 2: Expand the CSASPNETPagingGridViewMaintainCheckBox web application and press Ctrl &#43; F5 to show the Default.aspx.<br>
<span style=""><img src="/site/view/file/117879/1/image.png" alt="" width="163" height="336" align="middle">
</span><br>
Step 3: Select any Checkbox and then click paging links. Then return to the current page.<br>
Step 4: Validation finished.</p>
<h2>Using the Code</h2>
<p class="MsoNormal">Step 1: Create a C# &quot;ASP.Net Empty Web Application&quot; in Visual Studio. Name it as &quot;CSASPNETPagingGridViewMaintainCheckBox.&quot;<span style=""><br>
</span>Step 2: Add a web forms in the website root directory, name it as &quot;Default.&quot; Drag a GridView control to the page then convert one of its column as &quot;ItemTemplate&quot;, by putting CheckBox inside. Enable the GridView's Paging to true. The
 html code resembles following:</p>

<span style="">&nbsp;&nbsp;&nbsp;</span>List&lt;bool&gt; (The List&lt;bool&gt; should be saved into ViewState and it should contain as many values as the CheckBoxes' number)<span style="font-family:宋体">.</span>By using the (GridView.PageIndex*GridView.PageCount&#43; Row.RowIndex)
 to store state in CheckBox's CheckedChanged event. Then retrieve all the checkboxes' states from List&lt;bool&gt; to set Checkboxes states in Grid's RowDataBound event.<span style="">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
